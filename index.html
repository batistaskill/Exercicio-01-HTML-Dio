Tags

A tag <html></html> é a página como um todo, o elemento básico da estrutura do HTML. Ela conterá todos os elementos
do documento que será criado. Todo documento html deve iniciar com essa tag.

A tag <head></head> delimita o cabeçalho do documento, onde ficam as Não possui nenhum valor visível, porém é capaz de transmitir
ao navegador diversas informações muito úteis e essenciais a uma boa apresentação do documento HTML.

A tag <body></body> representa o corpo do documento, é nessa tag que todos os elementos visíveis do seu site devem ser inseridos.
